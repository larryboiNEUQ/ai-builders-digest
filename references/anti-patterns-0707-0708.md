# 7-07 / 7-08 与 7-14 差异速记

## 7-14（目标）
- 今日一句话 → 头条叙事（冲突/产品级）→ Builder 精选（可抄作业）→ 博客/播客 → 情绪板 → 判断
- 外链展开（latent.space 选型表）
- 评论原话短引 +「读法」
- 配图贴段落
- 标题 YYYY-MM-DD AI Builders Digest

## 7-07 / 7-08（要避免）
- 结构标题齐了，但正文是「每人：原文 + 说明」词条库
- 大量「编辑判断（非事实）」「样本中可见」「不代表共识」
- likes/views 精确到个位，像爬虫导出
- 外链只点名不拆内容
- 覆盖面全但无主线故事

## 一句话
读者应感到「今天行业在吵 X」；不应感到「我翻完了通讯录」。
